# Cordelius — Audit de producto · 2026-06-11

Base: rama `jarvis-ui-overhaul`, commit `56cc8ad`, modo stacked estable.

## Módulos existentes
`home`, `trading`, `health`, `journal`, `intelligence`, `alfredo` (Jarvis), `autopilot` — todos visibles en stacked mode, nav solo hace scroll.

## Datos VIVOS (reales)
- **WHOOP**: live vía OAuth (recovery, sleep, strain, HRV, RHR). Cache `whoop_today_cache.json` con refresh.
- **Noticias**: Finnhub (general + company news). ~30 items.
- **Quiver**: API configurada y respondiendo, pero congress/insiders/contracts = 0 filas hoy (LIVE pero vacío).
- **Journal**: 1 entrada real. Quick notes y checkins de Jarvis con pocos datos pero reales.
- **Portfolio history**: 600 puntos guardados.
- **Alerts**: 4 activas, pero generadas el 2026-06-08 → STALE.

## Datos FALSOS / SIMULADOS / MANUALES
- **Precios**: `refreshQuotes()` se invoca en boot y en un interval pero **nunca fue definida** → `corde.log` lleno de `refreshQuotes is not defined`. El objeto `quotes` solo se llena con pseudo-precios seeded (`stockPseudoAsset`, source `watchlist/research`). **Ningún precio del portafolio es live**: todo sale de `valueManual` hardcodeado en `PORTFOLIO`.
- **Indicadores** (RSI, MACD, momentum): explícitamente "SIMULADOS DETERMINISTAS" (seed por símbolo).
- **Régimen de mercado**: calculado sobre pseudo-quotes → en la práctica simulado.
- **Badge "Market Data: OK"**: engañoso — Finnhub solo alimenta noticias, no precios.
- **Bot paper**: explícitamente ficticio (cash simulado, trades simulados) — correcto pero sin badge SIMULATED visible.

## Duplicados
- Tarjetas de módulos renderizadas **dos veces** en Home (grid grande + mini-grid lateral).
- Rutas duplicadas (la segunda nunca se alcanza): `/api/whoop/status` (6035 y 6172), `/api/whoop/today` (6104 y 6198), `/api/journal/auto` (6136 y 6463), `/api/autopilot/database|progress|snapshot` (6073–6089 y 6648–6656).
- Valor del portafolio repetido 4+ veces en el primer scroll del Home.
- ~140 archivos `dashboard.backup.*`/`patch_*.js` en la raíz del repo (ignorados por git, pero ensucian el árbol).

## Números feos / vacíos / crudos
- `gainPct` interno con 15 decimales (la UI sí formatea con `money()`/`pct()`).
- Alertas muestran texto largo crudo de 3+ líneas.
- Quiver con 0 filas se ve "LIVE" sin aclarar que está vacío.

## UX confusa
- Home sin jerarquía clara: saludo → strip de stats → tarjetas grandes → brief → cards → pregunta de Jarvis → mini-grid duplicado.
- Meta-refresh de página completa cada N segundos (pierde estado de UI abierta).
- Input de Admin Token visible en el flujo normal (sección System).
- No hay forma rápida de actuar: todo es scroll.

## Mejorable sin reescribir
1. Definir `refreshQuotes()` real (Finnhub `/quote` para tickers USD) → arregla precios, régimen y el error en loop del log.
2. Badges honestos LIVE / FALLBACK / STALE / SIMULATED / MANUAL.
3. Command palette (⌘K) + acciones rápidas.
4. Panel "Jarvis Brain" ejecutivo arriba del Home.
5. Today Feed (timeline unificado WHOOP + portfolio + journal + news + autopilot).
6. Motor de reglas local (solo alertas educativas, sin órdenes).
7. Quitar duplicado de tarjetas en Home.
